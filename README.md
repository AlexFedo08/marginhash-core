# [MarginHash](https://marginhash.ru) Core Engine
Алгоритмическое ядро и математические модели платформы MarginHash для финансового анализа и прогнозирования доходности PoW-майнинга

**MarginHash Core** - это открытая часть аналитической платформы [MarginHash](https://marginhash.ru), содержащая математические модели и логику расчета финансовой эффективности майнингового оборудования (ASIC).

## О проекте
Цель данного репозитория - обеспечить прозрачность методологии расчетов, используемых на платформе. В отличие от стандартных калькуляторов доходности, MarginHash учитывает динамические рыночные переменные, что позволяет строить долгосрочные финансовые прогнозы.

## Ключевые функции (Math Logic)
* **Прогнозирование Халвинга:** Алгоритм автоматического пересчета доходности с учетом изменения награды за блок (включая прогноз на апрель 2028 года).
* **Индексация расходов:** Модель учета ежегодного роста тарифов на электроэнергию и обслуживание (OPEX).
* **Сценарный анализ:** Математическое обоснование «Пессимистичного», «Базового» и «Оптимистичного» сценариев окупаемости (ROI).
* **Нормализация данных:** Методология приведения технических характеристик 1000+ моделей ASIC к единому стандарту эффективности (J/TH).

## Технологический стек
* **Язык:** Python 3.10+
* **Библиотеки:** NumPy (для высокоточных расчетов), Pandas (для обработки временных рядов сложности сети).
* **Архитектура:** Модульная структура с разделением финансовой логики и слоев получения данных.

## Безопасность и прозрачность
В данном репозитории опубликована только **аналитическая и математическая составляющая**. Модули автоматизированного сбора данных (парсеры) и проприетарные алгоритмы защиты являются закрытой частью платформы для предотвращения несанкционированного использования ресурсов партнеров-поставщиков.
